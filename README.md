## 🚀 Installation et démarrage

### Prérequis
Assurez-vous d'avoir **Node.js** et **npm** ou **yarn** installés sur votre machine.

### Étapes pour démarrer le projet

1. Clonez ce dépôt :

   ```bash
   git clone https://github.com/Astray63/Portfolio.git
   cd Portfolio
   ```

2. Installez les dépendances :

   ```bash
   npm install
   ```

3. Démarrez le projet en mode développement :

   ```bash
   npm run dev
   ```

4. Construisez le projet pour la production :

   ```bash
   npm run build
   ```

5. Prévisualisez la version de production localement :

   ```bash
   npm run preview
   ```
